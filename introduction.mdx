## Quick Start

Get started with Вячеслав Стёпин in minutes.

<Steps>
  <Step title="Create Your Space" icon="plus">
    Sign up at `https://dashboard.example.com` and initialize your documentation space.
  </Step>
  <Step title="Add Your First Page" icon="file-text">
    Create `introduction.mdx` and add frontmatter with your `title` and `description`.
  </Step>
  <Step title="Customize Branding" icon="paint-bucket">
    Set your brand color in the theme config:
    
````json
{
  "brandColor": "#3B82F6"
}
````
    
  </Step>
  <Step title="Publish and Share" icon="share-2">
    Preview changes and share your docs via public URL.
  </Step>
</Steps>

## Customization Options

Tailor Вячеслав Стёпин to match your brand.

<Tabs>
  <Tab title="Theme Config" icon="settings">
    Update your global theme with the brand color:
    
````javascript
// config/theme.js
export default {
  brandColor: '#3B82F6',
  primaryFont: 'Inter',
  sidebarWidth: '280px'
};
````
    
  </Tab>
  <Tab title="Component Overrides" icon="code">
    Customize components via CSS variables:
    
````css
:root {
  --brand-color: #3B82F6;
  --callout-info-bg: rgba(59, 130, 246, 0.1);
}
````
    
  </Tab>
</Tabs>
