## Prerequisites

Before you begin, ensure you have:

- A modern web browser like Chrome, Firefox, or Safari
- An email address for account verification
- Internet connection

<Callout kind="info">

Вячеслав Стёпин requires account verification via email. Check your spam folder if you do not receive the confirmation email within 5 minutes.

</Callout>

## Sign Up and Log In

Follow these steps to create your account and access the dashboard.

<Steps>
  <Step title="Visit the Dashboard" icon="globe">
    Open your browser and navigate to `https://app.example.com`.
  </Step>
  <Step title="Create Account" icon="user-plus">
    Click **Sign Up** and enter your email, name, and a strong password.
  </Step>
  <Step title="Verify Email" icon="mail">
    Check your inbox for the verification link and click it to activate your account.
  </Step>
  <Step title="Log In" icon="log-in">
    Return to the dashboard, enter your credentials, and click **Log In**.
  </Step>
</Steps>

## Create Your First Documentation Space

A documentation space organizes your project docs. Create one now.

<Steps>
  <Step title="Navigate to Spaces" icon="book-open">
    In the sidebar, click **Spaces** > **New Space**.
  </Step>
  <Step title="Configure Space" icon="settings">
    Enter a name like `My Project Docs`, select a brand color (e.g., `#3B82F6`), and choose visibility (public or private).
  </Step>
  <Step title="Create Space" icon="check-circle">
    Click **Create**. Your space appears in the sidebar.
  </Step>
</Steps>

## Add and Edit Your First Pages

Add pages using the visual editor or MDX for advanced formatting.

<Tabs>
  <Tab title="Visual Editor" icon="edit-3">
    In your space, click **New Page**. Use the drag-and-drop blocks to add text, images, and components. Preview changes live and publish.
  </Tab>
  <Tab title="MDX Editor" icon="code">
    Select **MDX Mode** when creating a page. Write markdown with components.

    <CodeGroup tabs="Introduction,Code Example">
      ```mdx
      ---
      title: Welcome
      description: Your first page.
      ---

      ## Hello World

      This is your first MDX page.
      ```
      ```javascript
      // Example API call to fetch space data
      const response = await fetch('https://api.example.com/spaces/123', {
        headers: { Authorization: `Bearer ${YOUR_API_KEY}` }
      });
      const space = await response.json();
      console.log(space);
      ```
    </CodeGroup>
  </Tab>
</Tabs>
